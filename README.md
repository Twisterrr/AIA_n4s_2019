# Need4Stek
Epitech Project, Need4Stek

## How to use Need4Stek?
Execute "vrep.sh" to open V-REP and open a .ttt file in the software. Do "make" to create the binary "ia". Execute "pipe.sh" to execute the program and look V-REP.
